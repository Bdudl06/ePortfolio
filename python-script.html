Script Description

This script automates the process of reading a monthly sales CSV and calculating total revenue.

The Python Code

import csv

def calculate_sales(file_path):
    total = 0
    with open(file_path, 'r') as file:
        reader = csv.DictReader(file)
        for row in reader:
            total += float(row['Price'])
    print(f"Total Sales: ${total}")
